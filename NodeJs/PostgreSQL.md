## PostgreSQL

### 一. 安装
本机环境：Mac M1 [下载地址](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads) [安装教程](https://juejin.cn/post/6917861478926123021)

安装完成后默认会有一个账号： 
``` 
账号：postgres
密码：安装过程中设置的密码
```

将 Postgresql 路径添加到 PATH 环境变量：
```
vi /etc/profile

export PATH=${PATH}:/Library/Postgresql/11/bin

source /etc/profile
```
